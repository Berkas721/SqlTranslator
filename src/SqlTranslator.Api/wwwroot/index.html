О сервисе

Сервис автоматически преобразует SQL-запросы из диалекта PostgreSQL в диалект ClickHouse. Вставьте исходный запрос в левое поле и нажмите «Конвертировать».

В правом окне появится результат. Фрагменты, требующие внимания, выделены цветом — наведите курсор на фрагмент, чтобы увидеть пояснение.
